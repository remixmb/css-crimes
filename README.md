# CSS Crimes Lab

**A Stateful Application With No JavaScript and No Shame**

This is a fake app where CSS handles routing, permissions, incident severity, compliance review, pricing, observability, query planning, and narrative branching. None of it is JavaScript. All of it is a cry for help.

## Try the chaos combo

1. **DELETE /production-db**

Then add **Intern has prod access** + **Admin privileges, self-certified** for maximum RBAC astrology.

## What CSS is doing (and shouldn't)

- `@property` — typed variables that animate and calculate
- `calc()` — fake risk → fake latency, cost, blur, rotation
- `:has()` — state machine from checkboxes and radios
- `::before` — fake JSON, telemetry, compliance copy
- `@container` — layout logic from component width
- `color-mix()` — accent colors derived from “risk”
- `sin()` / `cos()` — orbital particle math
