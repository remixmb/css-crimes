What CSS is doing here


        

          Typed variables
          
@property defines numeric custom properties that can animate and participate in calculations.

        


        

          Math
          
calc() turns fake risk into fake latency, fake cost, blur, rotation, and meter width.

        


        

          State machine
          
:has() lets CSS inspect checked inputs and rewrite variables higher up the tree.

        


        

          Generated data
          
::before prints fake JSON, status codes, telemetry, and narrative output.

        


        

          Container logic
          
@container changes the component based on its own width instead of the viewport.

        


        

          Color math
          
color-mix() derives UI colors from the current “risk” state.
